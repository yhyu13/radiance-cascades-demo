## 📊 文档结构特点

### 视觉化优先

每个文档都包含大量图表：

```mermaid
mindmap
  root((文档特色))
    流程图[Flowcharts]
      数据流
      算法步骤
      管线架构
    序列图[Sequence]
      CPU-GPU 交互
      Uniform 传递
      纹理采样
    象限图[Quadrant]
      性能对比
      算法选择
      质量权衡
    时序图[Timeline]
      动画过程
      多 pass 执行
```

### 中英结合

- **技术术语**: 保留英文原文（方便查资料）
- **解释说明**: 使用中文（便于理解）
- **代码注释**: 双语标注

例如：
```glsl
// Uniform: 模型 - 视图 - 投影矩阵
uniform mat4 mvp;  // Model-View-Projection matrix

// SDF 圆形函数 - Signed Distance Field
bool sdfCircle(vec2 pos, float r);
```

## 🎓 教学理念

### "做中学" (Learning by Doing)

```
传统教学:
理论 → 演示 → 练习 → 项目
(被动接受)

我们的方法:
好奇 → 探索 → 实践 → 理解
(主动发现)
```

### "可视化优先" (Visualization First)

```
文字描述 ←→ 图表展示 ←→ 代码实现
     ↖________↙________↗
          相互印证
```

### "错误是朋友" (Embrace Mistakes)

```
犯错 → 调试 → 理解 → 成长
  ↑___________________|
       循环迭代
```
